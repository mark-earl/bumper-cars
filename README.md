# Bumper Cars

This program uses semaphores and pthreads to simulate a bumper car ride at an amusement park.

## Build Instructions
1. Ensure that you are in the source directory
```bash
/bumper-cars$
```
2. `make`
```bash
/bumper-cars$ make
```

3. `./bumper-cars.out`
```bash
/bumper-cars$ ./bumper-cars.out
```
